# Currency Converter

Android Application that allows conversion of different currencies with one another

## I. Installation of Android Studio

1. Download the latest version of Android Studio here: **https://developer.android.com/studio/preview**
2. Install Android Studio after the download has finished (Recommended version is Android Studio Narwhal)
3. Clone the repository and open it using Android Studio

## II.  Running the App
1. Import the project using Android Studio and wait for all of the dependencies to be downloaded.
2. Run the app by clicking the play icon

## IV. Features
1. The application demonstrates the retrieval of currency exchange rates from an API Service
2. The application provides mechanism to convert a selected currency to another one
3. The application provides a visual representation of all the balances for each currency
4. The application also demonstrates auto refresh  at a given time interval
5. The application show a visual representation of the current state of the application (Loading, Refreshing and Offline)

## V. Libraries / Technologies / Architecture
1. The architecture and design pattern used are **Clean Architecture** and **MVI** (Model-View-Intent) 
2. The User Interface of the app is built using **Jetpack Compose**
3. The network connection library used within the application is **Retrofit**
4. The dependency injection library used is **Hilt**
